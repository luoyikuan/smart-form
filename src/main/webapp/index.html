为移动而生

Smartform 采用业内先进的 mobile first 理念，从小屏逐步扩展到大屏，最终实现所有屏幕适配，适应移动互联潮流。

组件丰富，模块化

Smartform 含有 input 标签 文本、密码、邮箱、数字、单选、复选。 轻松完成表单的创建，一分钟开始数据收集的工作。

数据分析

Smartform 能帮您图形化来展示繁琐的数据。 让您一分钟成就数据分析师。

数据导出

Smartform 可导出 *.xlsx、 *.json、 *.xml、 *.txt 多种格式的文件。 覆盖了日常几乎绝大部分场景。
